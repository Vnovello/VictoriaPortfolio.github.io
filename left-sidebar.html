Project Details

User Experience

For the user experience I used a relational database such as MySql to register and manage all users in the platform.

Full Stack aplication

I developed this project with the idea that the user can use this portfolio as a template where they can fill out their experience, education, images and skills, using my aplication as their own portfolio.

The backend part of this project is developed using object-oriented Java, Springboot and MySql. I wrote the code myself and the business logic aswell.

For the front-end i focused on working with Angular as my framework, HTML and CSS primarily, but i explored the functionalities of Bootstrap. I was also able to rely on some pre-made templates that were developed with JavaScript wich i have some experience with.
